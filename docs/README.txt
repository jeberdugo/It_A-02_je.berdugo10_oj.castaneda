Pasos para instalar la base de datos
(Las querys usadas por estos pasos se encuentran en la carpeta data del proyecto):
1. Ejecutar las querys de limpieza de la base de datos, bien sea las encontradas en el
EsquemaAlohAndes.sql o sus propias sentencias si existen tablas no pertenecientes a este
proyecto
2. Ejecutar EsquemaAlohAndes.sql para crear las tablas a usar
3. Ejecutar poblacionTablas.sql para actualizar la base de datos con un escenario de mas de
5,000 filas.
